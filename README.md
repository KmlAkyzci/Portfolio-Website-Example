# 🚀 Creative & Responsive Portfolio Template

Modern, minimal ve tamamen responsive (mobil uyumlu) bir Kişisel Portfolyo / Şablon web sitesi. Bu şablon; geliştiricilerin, tasarımcıların veya içerik üreticilerinin projelerini ve yeteneklerini şık bir arayüzle sergilemesi için **HTML, CSS ve JavaScript** ile sıfırdan, temiz kod prensipleriyle geliştirilmiştir.

---

## ✨ Özellikler

* 🎯 **Kişiselleştirilebilir Profil Alanı:** Şablonu kullanan kişinin kolayca kendi görselini ve biyografisini ekleyebileceği esnek altyapı.
* 💻 **3x2 Modern Proje Galerisi:** Projelerin/web sitelerinin şık mockup kartları halinde listelendiği, esnek ve simetrik Grid düzeni.
* 🔄 **Sonsuz Kayan Yetenekler Barı:** Fare ile üzerine gelindiğinde duran, hem sürükleyip çekilebilen (drag-to-scroll) hem de kendi kendine kesintisiz akan modern Slider yapısı.
* 📱 **Tam Mobil Uyum (Responsive):** Masaüstü, tablet ve telefon ekranlarında düzeni bozulmadan kusursuz sıralanan CSS mimarisi.
* 🎨 **Göz Yormayan Renk Paleti:** Premium hissi veren koyu kartlar ile soft krem arka plan kontrastı.
